hexo主题的更换

发表于 2019-07-23 | 分类于 Hexo | | 阅读次数:

在搭建好自己的博客后会使用默认的landscape主题，是不是丑爆了，这次来说下我是如何更换主题的

主题的选择

现在流行的主题有yilia、next等。我们可以去hexo主题官网去挑选自己喜欢的主题风格。里边收录了很多牛人的主题。

选择一个你喜欢的主题，每个主题在GitHub上面都有开源的代码。下边我选择yilia主题为例来介绍

下载和切换主题

首先我们进入到你的本地博客目录下的themes，克隆你选择的主题从GitHub仓库到本地

下载完成后在themes就可以看到你新下载的主题
然后修改你博客目录下的_config.yml文件里的theme配置来更换使用的主题

然后，重新编译，启动

主题的配置

yilia主题就在themes目录下了，yilia目录下的_config.yml主要是一些主题配置，可以根据自己的需要修改

主题的更换和基本配置就到这里了 更详细的配置以后会专门再开一栏
